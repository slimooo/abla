FAQ

Est-il possible de tester gratuitement Abla ?
Oui. Vous pouvez utiliser sans aucun engagement Abla pendant 30 jours.
Abla est-elle compatible avec Wordpress, Wix ou Shopify ?
Oui. Abla est compatible avec tous les CMS et toutes les plateformes de publication. Il suffit d’ajouter une simple ligne de code dans la partie “head” du code source de votre site.
Peut-on intégrer Abla sur une application mobile ?
Oui. Abla fonctionne aussi bien sur iOS, Android ou sur le web mobile.
Est-il possible de continuer à utiliser un autre outil de mesure d’audience ?
Oui. Vous pouvez continuer à utiliser votre solution de mesure d’audience actuelle.
Abla utilise-t-elle des cookies ?
Non. Nous utilisons un script javascript pour le suivi. Cela nous permet de proposer une solution analytics sécurisée, moderne et respectueuse de la vie privée des visiteurs.
Les données sont-elles transférées ou vendues à des tiers ?
Non. Les données collectées via l’outil de mesure d’audience appartiennent uniquement à nos clients. Aucun transfert et aucune utilisation détournée ne sont réalisés.
Où sont hébergées les données ?
Les données sont hébergées sur des serveurs en France. Notre hébergeur est certifié ISO 27001, ISO 5001 et bénéficie de la qualification d'hébergeur de données de santé.
Est-il possible pour le visiteur de refuser le suivi d’audience ?
Oui. Nous mettons à votre disposition un script que vous pouvez ajouter sur votre site. Il permet à votre visiteur de demander la suppression des données collectées sur sa navigation et de refuser le suivi ultérieur.
Est-il possible d’ajouter des fonctionnalités personnalisées ?
Oui. Vous avez la possibilité de demander des développements supplémentaires et spécifiques à votre besoin, nous vous ferons une proposition commerciale adaptée.
Est-il possible d’obtenir des mesures sur autant de sites que l’on souhaite ?
Oui. Les offres Abla Analytics et Abla Analytics Plus incluent respectivement les analyses pour un et cinq sites. Par option, il est possible d’ajouter autant de sites que l’on souhaite pour un tarif de 4 euros HT par mois et par site.
Quelle est la différence entre une instance web mutualisée et individuelle ?
Par défaut, l’application Abla Analytics est mise à disposition de nos clients via un seul serveur que l’on appelle instance web mutualisée. Avec Abla Analytics Plus, vous pouvez bénéficier d’un serveur propre à votre application pour renforcer les performances de l’application, limiter les blocages via AdBlock ou par garantir un meilleur niveau de sécurité.
Est-il possible de choisir le nom de domaine de l’application ?
Oui. Abla Analytics est proposée avec un nom de domaine par défaut. Avec la version Plus, vous avez la possibilité de choisir le nom de domaine de votre application. Cette fonctionnalité permet notamment de réduire les risques de détection de l’outil de mesure d’audience par les bloqueurs publicitaires.
Quel avantage à utiliser une adresse IP dédiée ?
L’utilisation d’une adresse IP spécifique à votre application vous permet de réduire les risques de détection par les bloqueurs publicitaires.
